#Bank Management System (C++)

A full-featured console-based bank management system written in C++.
This project simulates a real banking system with multiple modules including client management, transactions, and user authentication with permission control.

##Features

_Client Management

-Show all clients

_Add new client

_Update client information

_Delete client

_Find client by account number

_Transactions

_Deposit money

_Withdraw money

_Show total balances

###User Management

_Add new users

_Update users

_Delete users

_Find users

##Authentication & Security

_Login system (username & password)

_Permission-based access control

_Admin and custom users

_System Modules

_Clients Management System

_Transactions System

_Users Management System

_Authentication System

##Data Storage

.Data is stored in text files:

_Clients.txt

_Users.txt

_Each record is saved using a custom delimiter.

##Concepts Used

.Structs

.Enums

.Bitwise permissions

.File handling (fstream)

.Vectors

.Multi-level menu system

##How to Run

-Compile the program:
g++ main.cpp -o bank
_Run the program:
./bank

###Example Flow

_Login using username and password

-Access system based on permissions

_Manage clients or perform transactions

_Logout
